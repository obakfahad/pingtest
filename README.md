# PINGTEST

This script is to test VPS host Data center Test IP and sort the IP by the latency.


### How To

`wget https://git.io/vzCNi -O /tmp/pingtest.sh;chmod a+x /tmp/pingtest.sh;bash /tmp/pingtest.sh`
